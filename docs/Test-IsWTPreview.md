# Test-IsWTPreview

## SYNOPSIS

Test if running in Windows Terminal Preview

## SYNTAX

```yaml
Test-IsWTPreview [<CommonParameters>]
```

## DESCRIPTION

This simple command will test if you are running in Windows Terminal Preview.

## EXAMPLES

### Example 1

```powershell
PS C:\> Test-IsWTPreview
False
```
